# FXLayoutView

## Requirements
- iOS 9.0+

## Feature
- MainAxisAlignment
    - [x] Start
    - [x] End
    - [x] Center
    - [x] spaceBetween
    - [x] spaceAround
    - [x] spaceEvenly
- CrossAxisAlignment
    - [x] Start
    - [x] End
    - [x] Center
 
## Installation

### [Swift Package Manager](https://github.com/apple/swift-package-manager)

Create a `Package.swift` file.

```swift
let package = Package(
    dependencies: [
        .package(url: "https://github.com/feixue299/FXLayoutView.git", from: "0.1.1")
    ],
    // ...
)
```
